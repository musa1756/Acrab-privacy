Общие сведения

Приложение Acrab (أَقْرَبُ — aqrab, «ближайший») — это образовательное iOS-приложение для изучения арабского языка. Мы серьёзно относимся к вашей конфиденциальности и стараемся собирать минимум данных, необходимых для работы приложения.

Какие данные собираются

Локально на вашем устройстве:

  • Имя пользователя (по желанию)
  • Уровень арабского языка и учебные цели
  • История чата с ИИ-ассистентом
  • Прогресс обучения: выученные слова, XP, серии дней
  • Сохранённые флешкарты, темы, аудиозаписи, глаголы
  • Настройки и предпочтения

Через серверы (минимально необходимое):

  • Текст запросов к ИИ — для получения ответов чата и флешкарт
  • Текст и аудио — для синтеза речи (TTS)
  • Данные авторизации (Apple ID / Google)
  • Анонимные события использования — для улучшения приложения

Разрешения устройства

Камера и фотогалерея — используются только для добавления изображений к флешкартам по вашему запросу.

Использование ИИ и передача данных

Acrab использует сторонние ИИ-сервисы для обработки запросов. Перед отправкой данных во внешние сервисы вы уведомлены об этом данной политикой.

Передаваемые данные используются исключительно для формирования ответов в приложении. Мы не передаём персональные данные для обучения моделей или в маркетинговых целях.

Список ИИ-сервисов, получающих данные:

  • Google Gemini — ИИ-чат и генерация флешкарт
  • Mistral AI — синтез речи (TTS)
  • ElevenLabs — синтез речи для уроков Fusha Drill

Каждый сервис имеет собственную политику конфиденциальности.

Аналитика

Мы используем PostHog для сбора анонимной статистики использования (открытые экраны, ошибки). Данные не содержат персональной информации и не передаются третьим лицам.

Авторизация

Supabase используется исключительно для авторизации через Apple ID или Google. Пароли не хранятся в приложении. Данные авторизации защищены iOS Keychain.

Хранение и безопасность данных

Большинство данных хранится локально на вашем устройстве. Все сетевые запросы выполняются через HTTPS. Мы не продаём и не передаём ваши персональные данные третьим лицам в маркетинговых целях.

Ваши права

Вы вправе в любой момент:

  • Запросить доступ к своим данным
  • Потребовать исправления неточных данных
  • Потребовать удаления своих данных

Для этого напишите на musa1756@proton.me с темой «Удаление данных». Мы обработаем запрос в течение 30 дней.

Изменения политики

Мы можем обновлять эту политику. При существенных изменениях уведомим через приложение. Дата последнего обновления указана в начале документа.
