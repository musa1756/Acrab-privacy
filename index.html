Политика конфиденциальности

Последнее обновление: апрель 2026 г.

Общие сведения

Acrab (عَقْرَب) — iOS-приложение для изучения арабского языка. Мы серьёзно относимся к вашей конфиденциальности и стараемся собирать минимум данных, необходимых для работы приложения.

Какие данные собираются

Локально на устройстве:

  • Имя пользователя (по желанию)
  • Уровень арабского языка и учебные цели
  • История чата с ИИ-ассистентом
  • Прогресс: выученные слова, XP, серии дней
  • Флешкарты, темы, аудиозаписи, сохранённые слова
  • Настройки приложения

Через серверы:

  • Текст сообщений — для получения ответов от ИИ-сервисов
  • Данные авторизации — при входе через Apple или Google
  • Статус подписки

Разрешения устройства

  • Микрофон — только при голосовом вводе в «Практике». Записи не сохраняются на наших серверах.
  • Камера — только для OCR-распознавания арабского текста в «Ташкиле». Фото не передаются третьим лицам.
  • Распознавание речи — Apple Speech Recognition, обработка на устройстве.
  • Фотогалерея — только для сохранения изображений флешкарт по запросу пользователя.

Сторонние сервисы

Для работы приложения используются внешние сервисы:

  • Google Gemini API — ИИ-ответы в чате
  • OpenRouter — обработка текста (переводы, нормализация)
  • Mistral AI — синтез речи (TTS)
  • Supabase — авторизация и управление подпиской
  • YooKassa — обработка платежей

Текст сообщений передаётся ИИ-сервисам исключительно для формирования ответов и не используется в других целях.

Авторизация

Приложение поддерживает вход через Apple ID и Google. Мы получаем только идентификатор пользователя и, при наличии разрешения, имя. Пароли не хранятся в приложении. Данные авторизации защищены iOS Keychain.

Хранение и безопасность

Большинство данных хранится локально на вашем устройстве. Все сетевые запросы выполняются через HTTPS. Мы не продаём и не передаём персональные данные третьим лицам в маркетинговых целях.

Дети

Приложение не предназначено для детей до 13 лет. Мы сознательно не собираем данные о детях.

Изменения политики

Мы можем обновлять эту политику. При существенных изменениях уведомим через приложение. Дата последнего обновления указана вверху страницы.
